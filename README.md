Robot Cam remote control
=====================

Remote control web interface for Robot based on Raspberry pi and Picon Zero hat.

add on hardware : 
- raspberry pi camera
- neopixels stick
- hc-sr04
- zumo chassis


Thanks to :
[WebControlledRobot]https://github.com/JamesPoole/WebControlledRobot
